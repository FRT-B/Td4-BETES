Contexte

Le projet BETES est réalisé dans le cadre de l'UE GLP Le projet est développé en Java et combine modélisation, programmation et interface graphique, permettant de simuler un environnement interactif et dynamique.

Objectif principal

Le but du projet BETES est de créer une simulation d'un mon vivant où des agents autonomes, appelés bêtes, évoluent de manière indépendante dans une grille NxN.
