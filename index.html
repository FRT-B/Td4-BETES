Description du projet :

Simulation d'un écosystème multi-agents en Java

Ce projet a été faite dans le cadre du cours de GLP

L’objectif du projet BETES est de simuler un écosystème artificiel composé d’agents autonomes, afin d’observer l’évolution des populations et les interactions entre les espèces. Les bêtes peuvent se déplacer, se nourrir, se battre et se reproduire au cours du temps.

Fonctionnalités principales :

La simulation inclut :

  • Une simulation 2D sous forme de grille
  • Des agents autonomes (les bêtes)
  • Un cycle jour/nuit influençant les comportements
  • Un système de combat, de reproduction et de consommation de nourriture
  • Une interface graphique permettant de suivre les déplacements des bêtes et l’évolution de la population
  • Un journal à la première personne pour chaque bête

Informations

  • UE: Génie logiciel et Projet (GLP)
  • Langage: Java
  • Types : Simulation multi-agents
  • Année: 2025-2026

Pourquoi ce projet ?

Ce projet permet de mettre en pratique la programmation orientée objet, la conception logicielle, ainsi que la modélisation de comportements autonomes dans un environnement dynamique. Bien qu’il ne représente pas parfaitement la réalité, il permet néanmoins d’en visualiser certains aspects.
