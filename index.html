Description du projet :

Simulation d'un écosystème multi-agents en Java

Le projet BETES est un logiciel de simulation qui permet d’observer l’évolution d’un ensemble d’espèces de petites bêtes dans différents types d’environnements. Les bêtes peuvent se déplacer, se nourrir, se battre et se reproduire au cours du temps.

Fonctionnalités principales :

La simulation inclut :

  • Une simulation 2D sous forme de grille
  • Des agents autonomes (les bêtes)
  • Un cycle jour/nuit influençant les comportements
  • Un système de combat, de reproduction et de consommation de nourriture
  • Une interface graphique permettant de suivre les déplacements des bêtes et l’évolution de la population
  • Un journal a la première personne de chaque bete
