## Description du projet
Ce site web sert de vitrine pour notre projet **"Bêtes"**. 
Il s'agit d'une simulation d'écosystème où des agents autonomes interagissent sur une grille.

### Fonctionnalités techniques
Le site respecte les contraintes du TD4 :
* Structure **XHTML5** valide.
* Utilisation des balises sémantiques.
* Navigation interne et externe.

### Structure du site
Page / Contenu

`index.html` / Accueil et présentation globale
`univers.html` / Univers de la simulation
`fonctionnalités.html` / Les fonctionnalités du projet
`combat.html` / Algorithme de combat
`architecture.html` / Patron MVC et Moteur/IHM
`equipe.html` / Présentation des membres
